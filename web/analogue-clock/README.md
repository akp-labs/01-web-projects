# Analogue Clock

Analogue clock made up of pure HTML, JS and CSS.
The UI is clean and minimal.
The background is light coloured with black, white and a bit red used for clock.



### Main features:
    * Hover effects are added.
    * Colour scheme is designed beautifully.
    * A small name plate on the top inside the clock.
    * Rounded shape of the clock made by pure JS.
    * Simple JS program used for hand movement.
    * Uses the time fetched by Java Script.
