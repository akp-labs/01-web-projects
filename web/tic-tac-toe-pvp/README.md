# Tic Tac Toe PvP

This is the two player version of the Tic Tac Toe Bot game. Two player can play this game together. The UI is dark themed with black background. The design is responsive on small screens.
If you want to play against bot, you may visit the repository mentioned below:
        *** tic-tac-toe-bot ***


## Main features:
    * Responsive design for different screens.
    * Progress bar to track the progress of both the players.
    * Easy interface for players.
