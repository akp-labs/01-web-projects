# Height Converter

Height converter(or length converter) is a very simple type of converter that can convert height from nearly any unit to any other unit(almost).
Though some units may be missing, they will be added after an update of this project.
Greenish colour is used on a white background giving it a very light feel.

## Main features:
    * Interface is very easy.
    * Doesn't perform calculation if required queries are missing.
    * Same feel on both phone and PC.
