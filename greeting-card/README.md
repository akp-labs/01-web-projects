# Greeting Card

A placeholder greeting card with placeholder images that can be used to design a greeting card for any purpose.
The card can be redesigned completely, giving the look of a completely new card.
The card can be downloaded as a PNG image and sent easily to anyone.


### Main features:
    * The placeholder card gives an idea of what can be designed.
    * You can add two images from your device gallery.
    * HTML2CANVAS is used for screenshotting the card and download it- very easy download.
    * The same can be redesigned and used multiple times.
    * Placeholder inspired from the Happy birthday greeting card from the repo `00-small-projects`
